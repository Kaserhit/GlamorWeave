# GlamorWeave

GlamorWeave syncs transmog outfits across your alts using WoW Retail's native Outfit Collection.

It syncs:

- name
- icon
- situations

## English

### Requirements

- WoW Retail 12.0 / Midnight
- Blizzard's native Transmogrification UI

### Quick Start

1. Open Transmog on your source character.
2. Click `Save From This Character`.
3. Open Transmog on your alt.
4. Click `Apply To This Character` or `Apply Slot`.
5. Optionally assign a keybind for `Wardrobe`.

### Features

- Syncs outfit name, icon, and situations
- Wardrobe panel for quick access
- Import / Export support
- Blizzard keybinding integration

### Notes

- Some actions still depend on Blizzard's protected UI.
- Direct outfit activation from a custom list may fall back to Blizzard's native flow.

- Интеграция с клавишами Blizzard

## Slash Commands

Main command:

`/gw`

Also available:

- `/glamorweave`
- `/osync` as a legacy compatibility alias

Useful commands:

- `/gw ui`
- `/gw quick`
- `/gw saveui`
- `/gw applyall`
- `/gw apply1`
- `/gw export`
- `/gw import <text>`
- `/gw dev`

## Compatibility

- WoW Retail 12.0 / Midnight
- Up to 20 synced outfit slots
